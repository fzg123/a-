# 生命游戏

## 简介

> 在游戏的进行中，杂乱无序的细胞会逐渐演化出各种精致、有形的结构；这些结构往往有很好的对称性，而且每一代都在变化形状。一些形状已经锁定，不会逐代变化。有时，一些已经成形的结构会因为一些无序细胞的“入侵”而被破坏。但是形状和秩序经常能从杂乱中产生出来。

## 亮点

- 使用ES6面对对象, 及单对象编程思想

- 使用webpack作为资源打包工具，使用了babel对代码就行了降级处理

## 各项功能

- 生命游戏基本功能已实现

- 鼠标单击或者按住不放，放置细胞

- 鼠标移出，菜单隐藏

- 重置游戏

- 停止游戏

- 仅开始一次游戏
